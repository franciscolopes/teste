HALLOWEENMEMORYCHALLENGE

Projeto Final desenvolvido como requisito parcial para conclusão da disciplina Fundamentos de Web Design II, do curso de Tecnologia em Sistemas para Internet do Instituto Federal de Educação, Ciência e Tecnologia do Triângulo Mineiro. Aluno: Francisco Lopes da Cunha. Professor: Wilton de Paula Filho.
Este é um jogo da memória, formado por 20 cartas, em que cada uma apresenta uma figura em um dos lados. Cada figura se repete em duas cartas diferentes. No início do jogo, o tabuleiro mostra as cartas com as figuras voltadas para baixo. Clique em uma carta para virá-la, comece virando duas cartas. Caso as figuras sejam iguais, você ganha esse par e joga novamente. Se forem cartas diferentes, estas cartas são desviradas e você deve tentar outra vez. O jogo possui um cronomêtro, que marca o tempo que cada jogador leva para encontrar todos os pares. Para começar o jogo, clique no botão JOGAR
